## このゲームは？

あなたは国税局の査察部に所属する国税査察官となり、脱税者の居宅、オフィス等に裁判所からの許可状を手に、強制調査を実施します。

制限時間内に全てのアイテムを見つけ出すと、脱税者は観念し、脱税を認めます。

**注意)脱税は罰金や刑事罰を課される犯罪です。**
