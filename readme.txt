東京国税局査察部です

あなたは東京国税局査察部所属の国税査察官となり、脱税の証拠を押さえるため、脱税者の家に強制調査に入ることになりました。
果たして、あなたは脱税の証拠を押さえ、脱税者に罪を償わせることができるのか！？
※脱税は犯罪です。税金を払うだけでは済みません。罰金や懲役刑を科されることとなります。
